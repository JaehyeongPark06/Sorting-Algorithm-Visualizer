## About
A sorting algorithm visualizer built with Python and PyGame. Features the following 7 algorithms:
- Bubble sort 
- Heap sort 
- Radix sort 
- Selection sort 
- Insertion sort 
- Merge sort 
- Quick sort

## Settings
The program will automatically set to 
- *Ascending*
- *Bubble Sort*
